whitepages by jsgoecke

Ruby Library for the Whitepages API - http://www.whitepages.com

The Whitepages Gem provides a library for consuming the web services provided by Whitepages. More information on the Developer API may be found here. This API may be used to lookup people by name as well as reverse lookups based on phone number or address.

There are a couple of limitations to consider (refer to the Whitepages website for any changes):

  • Maximum of 1,500 calls per day per API key
  • Throttle set at 2 searches per second per API key

Dependencies

  • json_pure v1.1.2+

Install

sudo gem install jsgoecke-whitepages --source=http://gems.github.com/
